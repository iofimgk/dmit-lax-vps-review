## DMIT 是什么，LAX EB 是什么意思

先把概念搞清楚，省得一头雾水。

DMIT 是 2018 年在美国纽约注册的 VPS 服务商，华人背景，中文客服，支持支付宝付款。自有机房，自建网络，不是转卖，这点很重要——他们跟中国三大运营商有直接的资源对接，手里有 CN2 GIA、CMIN2、9929 这些高端线路资源。

他们家的产品线按机房分成洛杉矶（LAX）、香港（HKG）、东京（TYO）三个地方，每个机房再分三个系列：

- **Premium（Pro）**：旗舰线路，三网 CN2 GIA 双向优化，贵，稳
- **Eyeball（EB）**：中间档，回程走 CMIN2 + 9929，性价比款
- **Tier 1（T1）**：国际线路，无中国大陆优化，胜在便宜量大

LAX EB，就是洛杉矶机房的 Eyeball 系列。Tiny 是这个系列里最小的套餐。

全名写出来是 **LAX.AN5.EB.TINY**——AN5 指的是当前这批宿主机用的是 AMD EPYC 9005（Turin 架构）平台，比之前老款的 7443P 性能有明显提升。

## 线路是这个套餐最值得讲的地方

Eyeball 系列的核心卖点就是线路，不然你为什么不直接去买更便宜的普通美西 VPS。

LAX EB 的路由配置：

1. **电信**：去程 CN2 优化线路，回程 AS9929
2. **联通**：去程 CN2 优化，回程 AS9929
3. **移动**：去程 CMIN2，回程 CMIN2

三网回程都有优化。移动用户回程走 CMIN2（AS58807），这个线路在同价位不太容易找到。

实测下来，从国内到洛杉矶这段，延迟大概在 150-170ms 区间，晚高峰期间（20:00-23:00）这个数字基本能保持住，不会出现那种白天测 150ms、晚上直接飙到 300ms 的情况。丢包率低，这是 CMIN2 线路的特点之一。

## 它到底适合哪些人

说实话，这个套餐不是给所有人的。来梳理一下：

**适合的场景：**

- 个人开发者跑小项目、博客、工具站，需要国内可以访问
- 偶尔要访问大陆用户，对延迟有一定要求但预算有限
- 需要美国原生 IP，用来解锁 Netflix、ChatGPT 这类服务（EB 系列用过的人通常提到解锁效果不错）
- 搭建代理节点，对线路质量有要求但不想付 Pro 系列的价格

**不太适合的场景：**

- 主要业务在大陆，对延迟极度敏感——那就直接上 LAX Pro（CN2 GIA 双向）或者香港 Premium
- 需要大流量、高并发——Tiny 的 1 核 2G + 1500GB 流量，业务稍微上量就不够用了
- 对 IP 被墙高度敏感——Eyeball 系列 IP 被墙可以付费换（每 15 天一次免费，之后 $5/次），但不如 Pro 系列更换机制灵活

顺便提一嘴：如果你是联通用户，现在的 EB 线路回程走 AS9929，体感上比早期版本好了不少。
